# iCampus 宿舍报修系统设计

校园辅助系统界面 UI 设计。

围绕"宿舍报修不透明"这一痛点，设计了面向**学生、宿管阿姨、维修师傅**三类角色的移动端报修系统。

## 文件说明

| 文件 | 说明 |
|---|---|
| `icampus-repair-prototype.html` | 可交互的移动端高保真原型（三角色切换、表单提交、状态流转） |
| `card-based-prototype.html` | 手绘风格的低保真卡片用例模型（5 张卡片） |
| `images/` | 角色照片 |

## 快速查看

直接在浏览器中打开两个 HTML 文件即可：

- **交互原型**：`icampus-repair-prototype.html` — 点击顶部标签切换学生 / 宿管 / 师傅视角
- **卡片原型**：`card-based-prototype.html` — 每张卡片对应一个关键界面模块
